# Feature: Login / Logout

User Story: As an Astro employee, I want to log in to Easy App using my Astro email credentials so that I can securely access my personalized home page and available apps.

## Steps:

- Sign-In Page: User starts with a sign-in page that uses the current Astro employee email account. Button: Sign in with Astro.


- Enter Email: User enters Astro email account and clicks Next.


- Enter Password: User enters password for Astro email account and clicks Sign in.


- Home Page: After successful authentication, user lands on the home page showing profile details and available apps.


- Logout: User can log out from the home page.





# Feature: Announcements

User Story: As an Astro employee, I want to view announcements in Easy App so that I can stay updated with important company information.

## Steps:

- Access Announcements: User can tap the bell icon at the top right of the home page or scroll to the announcement section and tap 'Announcements'.

- View Announcement List: User sees a list of announcements displayed in chronological order.

- Open Announcement Details: User taps on an announcement to view its full details.

- Search Announcements: User can search for specific announcements using keywords.

- Filter Announcements: User can filter announcements by category or date to narrow down results.



# Feature: QR Wallet

User Story: As an Astro employee, I want to use the QR Wallet feature so that I can make payments, scan QR codes, and track my transactions easily.

## Steps:

- Access QR Wallet: User clicks on the 'QR Wallet' icon from the home page.

- Scan QR Code: User clicks on the Scan QR button to scan a provided QR code for payment.

- Show QR Code: User clicks on the Show QR button to display their own QR code for others to scan.

- View Payment History: User clicks on Payment History to view all past transactions.

- View Transaction Details: User clicks on a specific transaction to view detailed information.

# Feature: Eclaims

User Story: As an Astro employee, I want to access and manage my claims so that I can submit new entries and view claim-related information easily.

## Steps:

- Access Eclaims: User clicks on the Eclaims icon from the home page.

- View Eclaims Guide: User clicks the top right side bar to view the Eclaims user guide in a web viewer.

- Access Eclaims Options: User clicks on the plus sign to reveal three options: Out of Office, Health Screening Claim, New Entry.

- Submit Out of Office Claim: User selects Out of Office and fills in details.

- Submit Health Screening Claim: User selects Health Screening Claim and enters details.

- Submit New Entry: User selects New Entry and creates a new claim entry.

# Feature: AstroDesk

# Feature: Report Piracy

User Story: As an Astro employee, I want to report piracy so that I can help protect Astro’s content and prevent unauthorized distribution.

## Steps:

- Access Report Piracy: User clicks on the Report Piracy icon from the home page.

- Choose Report Type: User sees two options: Report from a commercial outlet or Report a website.

- Report Commercial Outlet: User enters details about the outlet and can view previous reports under History.

- Report Website: User enters the URL and details of the site pirating Astro content.

# Feature: Steps Challenge

User Story: As an Astro employee, I want to participate in the Steps Challenge so that I can monitor my daily steps and compare my progress with others.

## Steps:

- Access Steps Challenge: User clicks on the Steps Challenge icon from the home page.

- Fetch Step Data: The app retrieves step data from the user’s health records automatically.

- View Rankings: The app displays rankings and values for all users who joined the challenge.

- View Progress Graph: User can see a graph showing their step count over time.

- Navigate Historical Data: User can scroll down or click on dots in the graph to view previous days’ step data.




# Feature: Profile

User Story: As an Astro employee, I want to access and update my profile so that I can keep my personal information accurate and up to date.

## Steps:

- Access Profile: User clicks on the Profile icon from the home page.

- Update Profile Picture: User clicks the photo icon to upload a new profile picture or delete the current one.

- Edit Personal Details: User clicks the edit icon to update details such as preferred language, contact number, and office phone extension.

- View Home Care Information: User clicks the Home Care section to view their current home care details.

- View Updates: User clicks the Updates section to see recent changes or notifications related to their profile.



# Feature: Forgot/Unlock Password

User Story: As an Astro employee, I want to reset or unlock my account password so that I can regain access to Easy App when I am locked out.

## Steps:

- Access Forgot/Unlock Password: User clicks on the Forgot/Unlock Password icon from the home page.

- Redirect to Microsoft Page: The app redirects the user to a Microsoft page where they can unlock their account or reset their password.
